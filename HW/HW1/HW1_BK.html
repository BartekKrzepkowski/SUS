Walidator¶

PrzyjmujƒÖc poziom istotnosci rowny $\alpha = 0.9$, wyznaczƒô oszacowanie b≈Çƒôdu generalizacji $BG$ w zale≈ºno≈õci od wielko≈õci zbioru testowego $|T|$, dlatego, ≈ºe wielko≈õƒá zbioru testowego zwiƒÖzana jest z $\epsilon > 0$, zale≈ºno≈õciƒÖ $-ln(1-\alpha) = 2 * |T| * \epsilon$.
Wynika z tego, ≈ºe ustalajƒÖc $T$, przy wcze≈õniej ustalonymy $\alpha$, wyznaczony zostaje $\epsilon$ oraz b≈ÇƒÖd empiryczny $BE_{T}$.
Zadanie polega zatem na minimalizacji sumy b≈Çƒôdu empirycznego $BE_{T}$ i $\epsilon$
$P(BG < BE_{T} + \epsilon) \geq 1-e^{-2|T|\epsilon}$

def eps_value(alpha, test_size):
    return np.sqrt(-np.log(1 - alpha) / (2 * test_size))

def err_plot(clf, alpha, k, to_cycle=5):
    bottom = 1/X.shape[0]
    dependency = []
    for test_freq in tqdm(np.linspace(bottom, 1-bottom, k+2)[1:-1]):
        upper_bound = 0
        for _ in range(to_cycle):
            x_train, x_test, y_train, y_test = train_test_split(X, y, test_size=test_freq)
            sk_nb = clf.fit(x_train, y_train)
            upper_bound += 1-sk_nb.score(x_test, y_test) + eps_value(alpha, y_test.shape[0])
        dependency.append((y_test.shape[0], upper_bound/to_cycle))
        
    dependency = np.array(dependency)
    upped_bound_idx = np.argmin(dependency[:, 1])
    test_size = dependency[upped_bound_idx, 0]
    ge_upper_bound = dependency[upped_bound_idx, 1]
    plt.grid(True)
    plt.xlabel("test_size")
    plt.ylabel("empirical_error + epsilon_value")
    plt.scatter(dependency[upped_bound_idx, 0], dependency[upped_bound_idx, 1], c="k")
    plt.plot(dependency[:,0], dependency[:,1], )
    
    return int(test_size), eps_value(alpha, test_size), ge_upper_bound
